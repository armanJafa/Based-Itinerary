Detailed Description

The visitObj struct helps the itinerary to optimze its order.

This struct holds two variables. The first is a boolean that tells the optimizing algorithm if the stadium the struct is representing has already been visited by the algorithm. The second boolean tells the algorithm if the stadium is actually in the itinerary, and is valid to be optimized.
