Detailed Description

The based_on_second struct provides instructions for std::sort to compare only the second integer of integer std::pairs.

It does this by overloading the () operator

The only function that uses this is Data::addDist() since the adjacency list needs to sort its respective std::list whenever a new distance is added, such that the smallest distance is always at the front and the longest is at the back.
