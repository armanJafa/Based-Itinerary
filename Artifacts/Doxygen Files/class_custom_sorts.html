vector< pair< int, QString > > CustomSorts::InsertionSort ( vector< pair< int, QString > >  sortThese)

Using an insertion sort, alphabetize a vector of (integer/QString) pairs.

Parameters
sortThese: Vector of data to be sorted
Returns
Returns a sorted version of sortThese

This sort sorts based on comparing the QStrings of the (integer/QString) pairs. It basically alphabetizes the vector.

The integer of the (integer/QString) pair represents a stadNum.

Complexity: O(n^2)
