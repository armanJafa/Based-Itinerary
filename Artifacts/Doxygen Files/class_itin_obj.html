Detailed Description

This class is the object representation of a queued stadium within the itinerary.

This class represents a queued stadium within the itinerary. It also performs functions that will help a tourist plan a trip.

For example, when a tourist wants to add souvenirs from a particular stadium that's on the trip itinerary into the wishlist, this class object will store the wanted souvenirs and their quantities.
